# PHASE 0 - Project Initiation

## Tại sao project này tồn tại ?

Artifact cần tạo

```text
0.1 Project Charter
0.2 Product Vision
0.3 Stakeholder Map
0.4 Problem Statement
0.5 Success Metrics
0.6 Assumptions & Constraints

```

## Câu hỏi cần trả lời 

```text

1. App Olympic này sinh ra để giải quyết vấn đề gì?
2. Ai là người đang gặp vấn đề?
3. Hiện tại họ đang xử lý bằng cách nào?
4. Cách hiện tại đau ở đâu?
5. App này thành công khi đo bằng chỉ số nào?
6. Giới hạn của project là gì? Thời gian, nhân lực, công nghệ, phạm vi?

```

## Exit criteria

```text
- Chốt được mục đích sản phẩm.
- Chốt được stakeholder chính.
- Chốt được pain point chính.
- Chốt được success metrics.
- Chốt được scope ban đầu ở mức business.
```
